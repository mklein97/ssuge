Detailed Description

@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@ The Application class is our central hub. It initializes all of our managers,

Has access to the ogre objects, etc. and contains the main loop. Most of the real work is delegated to the various managers (ScriptManager, InputManager, etc.)
