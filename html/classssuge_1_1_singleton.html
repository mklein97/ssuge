Detailed Description

template<class T>
class ssuge::Singleton< T >

@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@ A Singleton is a class which only allows one instance of itself and allows controlled global access. To make an existing class into a singleton, do the following:

  1. Derive your class from Singleton<Foo> (where Foo is your class)
  2. Define the static msSingleton attribute, generally in foo.cpp like this template<> Foo* Singleton<Foo>::msSingleton = NULL
  3. (optional) Define a convenience macro to simplify the global access (I usually do this in foo.h) @@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
