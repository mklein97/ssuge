Detailed Description

S.S.U.G.E (SSU Game Engine) is a simple, but functional game engine. The Engine is written in C++ and supports lua scripting (in theory, you can make an entire game in lua. It also uses these libraries:

  • Ogre (for Rendering and rendering-related resource management and window-support)
  • SDL (for input-handling)
  • tinyxml2 (for scene parsing)
  • Lua (for scripting)
