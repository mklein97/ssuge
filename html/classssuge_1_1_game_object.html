Detailed Description

@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@ A THING in our game (bullet, player, trigger-point, etc.) Contains a collection of components that give it functionality. By itself, the game object is basically a node within a scene graph (and so has a position, orientation, scale, and the ability to child to another GameObject) and not much else. GameObject inherits from InputObjserver so a particular game object can be registered as a listener with the InputManager @@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
